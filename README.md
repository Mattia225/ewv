⚠️ Note on EWW
EWV (Enhanced Widget Vanguard) is NOT a fork, an extension, or a wrapper for eww.

While it supports .yuck syntax for configuration (to make life easier for widget developers), the core logic is entirely different:

Engine: Written in C for maximum performance and low memory footprint.

Launcher: Written in C++ to manage processes and PID files efficiently.

Architecture: Built specifically for Wayland using GTK4 and Layer Shell from the ground up.
